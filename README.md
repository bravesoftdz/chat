# Chat

## Installation
+ run the script ```"db.sql"``` in the folder migration
+ run command ``` composer dumpautoload -o```
  
## Usage package
+ pusher-php-server
+ predis

+ login & sign up
+ users list
+ friends list
+ friends request notification
